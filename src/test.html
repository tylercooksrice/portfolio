Nested Loop Week 7: Nested If Else

In this short demonstration, this video project shows an example of of using nested if else statements as two comparisons are done. The first if else statement determines if the user entered in Yes(Y) or No(N). Afterwards, if the user enters in Yes(Y), then a comparison of ages are done. If the user's randomly generated age is more than 7, then the penguin is allowed to swim, else the penguin is not allowed to swim. If the user enters in No(N) then the penguin will simply go to the beach to build a sand castle.
